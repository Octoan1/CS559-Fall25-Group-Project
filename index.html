Game Info

Controls
  • W / S / Up / Down: Tilt platform (forward/back)
  • A / D / Left / Right: Tilt platform (left/right)
  • R: Reset ball to current level start
  • N: (Debug) Teleport ball to the hole / skip level
  • G: Toggle grid lines (debug)
  • H: Toggle grid coordinates (debug)
Game Modes
  • Endless: Procedurally generated, endless levels. Objective: reach the hole.
  • Race: A timed, high-score variant of Endless mode where you must complete the level within a strict time limit.
  • Level: Pre-designed levels with specific challenges.

Tip: Toggle Dark Mode in Settings for an alternate look.
